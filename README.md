# zhao

a [Sails](http://sailsjs.org) application


#Instructions:

1. Install Sails - [Official Document](http://sailsjs.org/get-started)

	Example:
	
	```
	sudo npm -g install sails
	```

1. Install Ruby and SASS compiler

	Example:
	
	```
	gem install sass
	```
	
1. Install Dependencies (like every nodejs app)

	```
	npm install
	```
	
1. Let's Lift Sails

	```
	sails lift
	```
